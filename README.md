# MatrixData (C++)

Simple implementation of a matrix data game in the terminal. This works with a data matrix, when the player walks the code gaves info about the tiles around the player.

## How to run
Compile with g++:
g++ src/main.cpp -o game

Run:
./game

## What I learned
- game loop structure
- input handling
- basic collision logic

## Status
Early project. Not fully optimized.
